API Documentation

Access mosque data programmatically using our REST API.

Base URL

http://localhost:3000
GET/api/all

Returns all mosques as a JSON array

Example Request

fetch("http://localhost:3000/api/all")
  .then(res => res.json())
  .then(data => console.log(data));

Example Response

[
  {
    "id": "unique_id",
    "name": "Masjid Al-Sultan",
    "island": "Malé",
    "atoll": "Kaafu",
    "friday": true,
    "ladies": true,
    "capacity": 500,
    "latitude": 4.1755,
    "longitude": 73.5093
  }
]
GET/api/mosque?name={search}

Search mosques by name (case-insensitive, partial match)

Parameters

NameTypeRequiredDescription
namestringYesSearch query for mosque name

Example Request

fetch("http://localhost:3000/api/mosque?name=sultan")
  .then(res => res.json())
  .then(data => console.log(data));
GET/api/filter?{params}

Filter mosques with multiple conditions

Parameters

NameTypeRequiredDescription
atollstringNoFilter by atoll name
islandstringNoFilter by island name
fridaybooleanNoFriday prayer availability
ladiesbooleanNoLadies' section availability
capacity_gtnumberNoCapacity greater than value
capacity_ltnumberNoCapacity less than value

Example Request

fetch("http://localhost:3000/api/filter?atoll=Kaafu&friday=true")
  .then(res => res.json())
  .then(data => console.log(data));

Mosque Object Structure

Schema

{
  "id": "string",
  "name": "string",
  "island": "string",
  "atoll": "string",
  "dhivehi": {
    "name": "string (Dhivehi script)",
    "island": "string",
    "atoll": "string"
  },
  "friday": boolean,
  "ladies": boolean,
  "capacity": number,
  "latitude": number,
  "longitude": number,
  "buildDate": "string (ISO date)",
  "picture": "string (URL)",
  "description": "string",
  "contact": "string"
}
